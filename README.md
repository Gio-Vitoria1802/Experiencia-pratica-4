# Experiencia-pratica-4
Sistema de Gestão da Biblioteca Universitária – UFSP
Projeto desenvolvido para a Experiência Prática 3.

Objetivo
Implementar e manipular dados em um banco de dados relacional utilizando SQL (DML), aplicando 1FN, 2FN e 3FN, versionando o código e utilizando ferramentas de desenvolvimento como MySQL Workbench.
